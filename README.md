# 🔥 Network Security Architecture – pfSense (Firewall, NAT, VPN)

## 🎯 Overview
Implementação de arquitetura de rede segura com firewall pfSense, incluindo NAT, controlo de acessos e VPN.

---

## 🛠️ Tecnologias
- pfSense
- NAT
- Port Forwarding
- OpenVPN

---

## 🔎 Atividades Realizadas
- Configuração de interfaces WAN e LAN
- Implementação de NAT para acesso à Internet
- Configuração de Port Forwarding para publicação de serviços
- Restrição de acesso SSH por endereço IP
- Implementação de VPN (OpenVPN)
- Testes de conectividade e acesso externo

---

## 🚨 Resultados
- Publicação controlada de serviços internos
- Acesso remoto seguro via VPN
- Implementação de princípio de menor privilégio

---

## 📌 Notas
Projeto focado em segurança perimetral e controlo de acessos.
